== Description ==

Auto NoFollow Links adds nofollow attribute to all external links in your blog posts automatically.
save you a tremendous amount of time.

= Features =

*   Add rel="nofollow" to all external links, SEO friendly.
*   Add target="_blank", Open external links in new window or tab

== Installation ==

1. Login to your WordPress Dashboard.
2. Navigate to "Plugins" > "Add New".
3. Then click "Upload" in the header.
4. Choose the zip file you downloaded earlier.
5. Click "Install Now".
6. Once installed, tap on "Activate Plugin"
